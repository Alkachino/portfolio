# **App Name**: CodeCanvas

## Core Features:

- Hero Section: Display a clear introduction with the developer's name, title, and a brief summary. Includes prominent call-to-action buttons: 'View Projects' and 'Contact'.
- About Me Section: Showcase a professional story highlighting the developer's skills, passion, and experience. Includes education details and career goals.
- Skills Showcase: Visually attractive categorized display of technical skills and tools. Categories: Languages, Frameworks, Databases, DevOps & Containers, AI.
- Projects Gallery: Dynamic gallery to showcase featured projects. Each project includes a title, description, technologies used, and links to GitHub and live demos.
- Certifications & Formation Section: Highlight certifications and ongoing education, showing commitment to continuous learning and up-to-date skills.
- Contact Section: Provide contact information including email address and professional network links (LinkedIn, GitHub). Includes a final call-to-action button to download the CV.
- AI-Powered Content Improver Tool: Incorporate a tool using generative AI that allows the developer to enter any of the above written content and receive suggestions to improve it based on its context.

## Style Guidelines:

- Primary color: Desaturated dark blue (#343B48) to maintain a professional and tech-oriented look.
- Background color: Almost-black (#1E212B) to establish a dark mode theme.
- Accent color: Soft teal (#74B4A7) as a subtle highlight for interactive elements, links, and key information.
- Font Pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, ensures readability and modern appeal.
- Use clean, minimalist icons for skills and contact information.  All icons to be monochrome, matching the accent color.
- Use a clean, grid-based layout to showcase the content in an organized manner. Ensure the site is responsive and adapts to different screen sizes.
- Incorporate subtle animations on hover effects and transitions to improve user experience and add a modern touch.
